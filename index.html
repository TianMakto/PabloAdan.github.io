Dusty - El Palo Games

Dusty is a narrative and puzzles solving 3D experience where the player will control Dusty, an archeologist who is looking for a lost kingdom.

I'm working as a gameplay programmer for ElPaloGames in Unreal Engine. You can take a look at the trailer of the game! The game will be released soon!

My role involved various aspects of game development, including:

  • Creating most of the gameplay mechanics such as the pole jump and pole elevator
  • Managing the save system and level streaming for the game
  • Developing the UI and HUD elements
  • QA testing and bug solving

C++ 2D Game Engine

Personal project with the purpose of learing how an engine works while making my dream game (a mix between fire emblem and DnD), still working on it on my free time.

Some of the main tasks I already implemented on my engine are:

  • Logic Manager in charge of executing the tick of the game and all the logic with a entities system
  • Renderer Manager for painting in layers my different sprites
  • Entities and componentes system inspired by Unreal Engine
  • Modular UI, for easy creating new menus (never thought printing text would be not super easy)
  • Pathfinding for the game AI inspired on A* and adjusted to the game necessities
  • Integration of external tools for easying level designing since the engine lacks an editor (for the moment)
